# Selfhosted Bookmark Tools Research Bundle

Diese ZIP enthält eine strukturierte Recherche zu:

- Karakeep
- linkding
- Linkwarden
- LinkAce
- Readeck
- Shiori
- Proxmox Community Scripts

Inhalt:
- `FULL_REPORT.md` – ausführlicher Gesamtbericht
- `projects/*` – Einzelsteckbriefe pro Tool
- `tables/comparison.csv` – Vergleichstabelle
- `sources_urls.txt` – gesammelte Web- und GitHub-Links
- `LICENSES/*` – Lizenz- und Quellenhinweise
- `diagrams/*` – Mermaid-Diagramme
- `scripts/dedup-examples.md` – Beispielansätze für Dublettenbereinigung

Hinweis:
Der Bericht basiert auf der zuvor erzeugten Deep-Research-Sammlung. Wo in den Primärquellen keine belastbare Aussage verifiziert werden konnte, ist das explizit als „unklar“ oder „nicht belastbar bestätigt“ markiert.
